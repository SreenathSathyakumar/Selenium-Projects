# 🚀 Web UI Automation with Selenium, Java & TestNG

This is a demo UI automation project built using:

- 🧪 Selenium WebDriver
- 🛠️ Java + TestNG
- 📄 Page Object Model (POM)
- 📊 TestNG Report (HTML)
- 📸 Screenshot on failure

## ✅ Test Cases Covered:
- Login with valid/invalid credentials
- Product search flow
- Logout functionality

## 🔧 Tech Stack
- Java 11
- Maven
- Selenium 4
- TestNG
- Page Object Model

## 📁 How to Run
1. Clone the repo  
2. Run via TestNG XML (`testng.xml`)  
3. See reports under `reports/` folder
